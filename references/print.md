# PDF打印工具 print.py

## 简介

`print.py` 可以实现对 PDF 文件的静默打印。支持自定义打印页码范围、打印顺序、页面缩放等高级选项，适用于批量打印场景。

## 功能特点

- 支持批量打印多个 PDF 文件
- 可指定打印页码范围
- 支持奇偶页选择性打印
- 可控制页面缩放适应纸张
- 支持反向打印顺序
- PostScript Level 可选（2/3）

## 使用方法

### 基本语法

```PowerShell
python .\scripts\print.py <输入PDF文件1> [<输入PDF文件2> ...] [选项]
```

### 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `input_pdfs` | 位置参数 | 必填 | 要打印的 PDF 文件路径列表 |
| `--start` | 整数 | 0 | 起始页码（从0开始） |
| `--end` | 整数 | -1 | 结束页码（-1表示最后一页） |
| `--ps-level` | 整数（2/3） | 3 | PostScript Level 级别 |
| `--disable-shrink-to-fit` | 标志 | False | 禁用缩放页面到适合纸张（保持原大小） |
| `--reversed` | 标志 | False | 反向打印（从最后一页开始） |
| `--page-option` | 整数（0/1/2） | 0 | 页面选项：0=全部打印，1=打印偶数页，2=打印奇数页 |

### 使用示例

#### 1. 基本打印
```PowerShell
python .\scripts\print.py document.pdf
```

#### 2. 打印指定页码范围
```PowerShell
# 打印第3页到第6页（从0开始计数）
python .\scripts\print.py report.pdf --start 2 --end 5
```

#### 3. 打印所有 PDF 的奇数页
```PowerShell
python .\scripts\print.py doc1.pdf doc2.pdf doc3.pdf --page-option 2
```

#### 4. 反向打印并保持原大小
```PowerShell
python .\scripts\print.py manual.pdf --reversed --disable-shrink-to-fit
```

## 页面选项详解

| `--page-option` | 说明 |
|-----------------|------|
| 0 | 打印所有页面（默认） |
| 1 | 仅打印偶数页 |
| 2 | 仅打印奇数页 |

## 缩放选项说明

- **默认行为**：启用缩放，页面会自动缩放以适应纸张大小
- **使用 `--disable-shrink-to-fit`**：保持页面原始大小，可能超出纸张边界

## 注意事项

1. **打印机设置**：使用系统默认打印机，请确保默认打印机已正确配置
2. **页码范围**：页码从0开始计数，`--start` 和 `--end` 参数包含两端
3. **奇偶页打印**：与 `--start`/`--end` 参数配合使用时，会在指定范围内筛选奇偶页
4. **批量处理**：单个文件打印失败不会影响其他文件的打印任务
5. **静默打印**：不会显示打印对话框，直接发送到打印机

## 常见问题

### Q: 打印任务提交成功但没有输出？
A: 可能的原因：
   - 打印机缺纸或离线
   - 默认打印机设置有误
   - 打印队列被暂停
   - Acrobat 打印权限限制

### Q: 如何打印到指定打印机？
A: 当前版本使用系统默认打印机。

### Q: 页面被裁切或缩放不正确？
A: 根据需求选择：
   - 需要完整打印页面：使用 `--disable-shrink-to-fit`
   - 需要适应纸张：移除 `--disable-shrink-to-fit` 参数

## 性能提示

- 打印大文件（100页以上）时，建议先测试小范围打印
- 批量打印大量文件时，注意打印机缓存和纸张容量
